4 Printer Calibration

Printers can slightly stretch prints. Use these controls to ensure absolute millimeter correctness.

Calibration steps:

  1. Print the template with rulers enabled.
  2. Measure the vertical & horizontal rulers with a real physical ruler.
  3. If the printed Y-ruler shows 99mm instead of 100mm, divide actual by measured ($100 / 99 \approx 1.0101$) and set the Scale Factor.
